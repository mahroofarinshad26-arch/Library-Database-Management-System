# Library-Database-Management-System


## Project Description
Library Management System is a database project developed using MySQL. It helps manage library operations such as storing book details, customer information, employee records, book issue status, and return status. 

## Objectives

- To create a Library Management System using MySQL.
- To manage books, customers, employees, and branches.
- To keep track of issued and returned books.
- To use SQL queries for storing and retrieving data.
- To maintain data efficiently using a database.
## Database Name
library

## Tables Used

1. Branch
2. Employee
3. Books
4. Customer
5. IssueStatus
6. ReturnStatus

## Features

- Manage book details and availability status
- Store customer information
- Track issued and returned books
- Manage employee and branch details
- Generate reports using SQL queries

## Technologies used 

- MySQL
- SQL



### SQL Concepts Used
- Joins
- Aggregate Functions
- GROUP BY
- HAVING
- ORDER BY
- Subqueries
- LIKE Operator

## Queries Included

- Available books list
- Employee salary details
- Issued books with customer names
- Book count by category
- Employees with salary above ₹50,000
- Customers who have not issued books
- Employee count in each branch
- Customers who issued books in June 2023
- Books containing "History" in title
- Branches with more than 5 employees
- Branch managers and addresses
- Customers who issued books with rental price above ₹25



## Output

The project successfully demonstrates the implementation of a Library Management System using MySQL with multiple tables, relationships, and SQL queries.
## Conclusion

The Library Management System helps organize and manage library data efficiently. It provides an easy way to store, retrieve, and track information about books, customers, employees, issued books, and returned books using MySQL.
